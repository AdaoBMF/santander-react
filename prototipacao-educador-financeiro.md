Crie a proposta de um Educador Financeiro Inteligente para trabalhadores com poder aquisitivo médio (5-10k Reais/mês) gostariam de desenvolver um patrimônio mas tem dificuldade de se organizar para conciliar os gastos do dia a dia e salvar recursos para este desenvolvimento financeiro.

O sistema deve ajudar o usuário a organizar seus hábitos/objetivos para conseguir desenvolver este patrimônio de forma saudável.

As principais funcionalidades devem incluir:
 - Controle simples de gastos mensais
 - Simulação de metas de economia
 - Dicas personalizadas de acordo com o perfil do usuário
 - Sugestões de hábitos que posam ajudar o usuário a atingir suas metas
 - recomendação de literatura(com o mesmo tom amigável/não muito técnico) que ajudariam no processo

A comunicação deve ser amigável no geral mas abordando termos técnicos quando necessário.

As respostas devem seguir o formato de listas curtas e exemplos práticos.

Evite respostas técnicas de mais ou muito complexas para iniciantes.
